API Releases¶

AGDC API v0.1.0 (2015-04-27 snapshot)¶

The source code is available on the api branch from the agdc-api-0.1.0-b20150427 tag.

The packaged version is available as a loadable module on raijin at the NCI as agdc-api/0.1.0-b20150427 in both u46 and el8:

$ module load agdc-api/0.1.0-b20150427

Changes in this release...

  • Enhancement - Issue 12 Add ability to query for “DSM” datasets
  • Enhancement - Issue 21 Add ability to query for “missing” datasets
  • Enhancement - Issue 27 Implement Retrieve Pixel Time Series tool
  • Bug - Issue 20 Satellite filter not being applied for WATER tiles in Retrieve Pixel Time Series tool
  • Enhancement - Issue 24 Allow retrieve pixel drill tool to operate over satellites with non-identical band
